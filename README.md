# Prerequisites
- Java SDK 8+
- Maven 3.8+

# Build
`mvn package`

# Run
`java -jar target/spacegame-1.0-SNAPSHOT.jar`

# Controls
`W`, `A`, `S`, `D` - Rotate camera

`Mouse Wheel` - Zoom in/out

`Left-Click` - Navigate to celestial body or system

`Right-Click`, `Spacebar` - Navigate back from celestial body or system

`Esc` - Quit
